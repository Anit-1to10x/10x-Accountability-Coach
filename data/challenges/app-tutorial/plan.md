# App Tutorial - 7-Day Plan

## Overview
Learn how to use the 10X Accountability Coach app effectively. Master all features including challenges, check-ins, streaks, and the AI coaching system.

## Week 1: Complete App Mastery

### Day 1: App Overview & Navigation
- [ ] Explore the left sidebar navigation
- [ ] Visit the Calendar/Schedule page
- [ ] Check out the Challenges page
- [ ] Browse the Todos section
- [ ] Open Settings and review options

### Day 2: Creating Your First Challenge
- [ ] Configure challenge settings (duration, daily commitment)
- [ ] Review your challenge dashboard
- [ ] Explore the Challenges page
- [ ] Understand challenge types (habit, learning, building, creative)
- [ ] Create a test challenge with your own goal

### Day 3: Daily Check-ins & Streaks
- [ ] Complete your first daily check-in
- [ ] Review the streak tracking system
- [ ] Set up your check-in reminders
- [ ] View your check-in history
- [ ] Understand the grace period system

### Day 4: Using the AI Coach
- [ ] Start a conversation with the AI coach
- [ ] Ask for help with your challenge
- [ ] Try different coaching modes
- [ ] Explore agent personalities
- [ ] Set your preferred coaching style

### Day 5: Calendar & Schedule Management
- [ ] Add a task to your calendar
- [ ] View different calendar views (day, week, month)
- [ ] Schedule time blocks for your challenge
- [ ] Set up recurring tasks
- [ ] Review your weekly schedule

### Day 6: Todos & Task Management
- [ ] Create a new todo item
- [ ] Set priority levels on tasks
- [ ] Mark tasks as complete
- [ ] Organize tasks by category
- [ ] Review your task completion history

### Day 7: Advanced Features & Tips
- [ ] Explore the vision board feature
- [ ] Review accountability contracts
- [ ] Check cross-challenge insights
- [ ] Customize your workspace
- [ ] Plan your next challenge

## Milestones
- [ ] Complete app orientation (Day 1)
- [ ] Create first custom challenge (Day 2)
- [ ] Achieve 3-day streak (Day 3)
- [ ] Master all core features (Day 7)

## Success Criteria
- Navigate all app sections confidently
- Create and manage challenges independently
- Maintain a daily check-in habit
- Use the AI coach effectively
